# 🚀 QA Automation Project - Madar System

This repository contains the automated testing suite for the **GeoVerse/Madar** project. The goal is to ensure high quality and reliability through professional testing methodologies.

## 🛠 Tech Stack
- **Language:** TypeScript
- **Framework:** Jest / Playwright
- **CI/CD:** GitHub Actions
- **Environment:** Node.js

## 📂 Project Structure
- `tests/`: Contains all automation test scenarios.
- `.github/workflows/`: CI/CD pipeline configuration (runs tests on every push).
- `src/`: (Optional) Core logic and Page Object Models.
